# Projek Pemrograman Web I - SIFU

## Deskripsi Proyek
SIFU adalah website interaktif yang mengimplementasikan sistem dan mekanik tertentu
sesuai dengan tema yang ditentukan pada Responsi 2 mata kuliah Pemrograman Web

## Tema Proyek
Sistem Gamifikasi Pengembangan Karakter (SIFU)

## Fitur Utama
- Sistem interaksi user
- Perubahan status berdasarkan aksi user
- Tampilan visual yang disesuaikan dengan tema
- Penggunaan JavaScript dan backend untuk interaksi dinamis

## Role User
- User

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- PHP

## Cara Menjalankan Proyek
1. Clone repository ini
2. Jalankan di localhost (XAMPP / Laragon)
3. Buka melalui browser

## Catatan
Proyek ini dibuat untuk memenuhi tugas Responsi 2
